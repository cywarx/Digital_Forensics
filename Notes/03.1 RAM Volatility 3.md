
---
### Step 1: 
From your actual memory dump — run this: Export pstree data to the json format

```bash
vol -f memdump.mem -r json windows.pstree > pstree.json
```

---
### Step 2: 

**Python File: (Vol3_dot.py)**
![[vol3_dot.py]]

#### **Usage:**

```bash
# From saved JSON
python3 pstree2dot.py -i pstree.json -o pstree.dot

# Piped directly from Volatility
vol -f memdump.mem -r json windows.pstree | python3 pstree2dot.py -o pstree.dot

# Render
dot -Tsvg pstree.dot -o pstree.svg
xdot pstree.dot
```

#### With Path
```bash
# From saved JSON
python3 pstree2dot.py -i /tmp/pstree.json -o /tmp/pstree.dot

# Piped directly from Volatility
vol -f memdump.mem -r json windows.pstree | python3 pstree2dot.py -o /tmp/pstree.dot

# Render
dot -Tsvg /tmp/pstree.dot -o /tmp/pstree.svg
xdot /tmp/pstree.dot
```

---

**Set it up (Tool):**

```bash
# Rename and move to a tools directory
mkdir -p ~/tools
mv pstree2dot.py ~/tools/pstree2dot.py

# Make executable
chmod +x ~/tools/pstree2dot.py

# Add shebang is already there — now add to PATH in zshrc
echo 'export PATH="$HOME/tools:$PATH"' >> ~/.zshrc
echo 'alias pstree2dot="python3 $HOME/tools/pstree2dot.py"' >> ~/.zshrc

# Reload
source ~/.zshrc
```

**Now use it from anywhere:**

```bash
# From saved JSON
pstree2dot -i /tmp/pstree.json -o /tmp/pstree.dot

# Piped directly
vol -f memdump.mem -r json windows.pstree | pstree2dot -o /tmp/pstree.dot

# Full one-liner
vol -f memdump.mem -r json windows.pstree | pstree2dot -o pstree.dot && xdot pstree.dot
```
